# Installation 🌈

1. Clone the repository with the magic of git:
   ```bash
   git clone https://github.com/GabrielBarrueco/PokeDex-study-project.git
   ```

2. Conjure the dependencies:
   ```bash
   yarn install
   ```

3. Begin the journey on the development server:
   ```bash
   yarn start
   yarn ios
   yarn android
   ```
